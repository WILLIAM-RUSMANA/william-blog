EternalBlue is one of the most significant exploits in modern cybersecurity history — an NSA-developed exploit for a vulnerability in Microsoft's SMB protocol (MS17-010) that was leaked by the Shadow Brokers group and later weaponized in the WannaCry and NotPetya ransomware attacks.

The practical walked through exploiting a vulnerable target using Metasploitable (a deliberately vulnerable Linux VM) as the test environment. The EternalBlue exploit targets Windows machines with unpatched SMBv1, allowing remote code execution without authentication.

The key lesson: publicly disclosed exploits and PoC code can be rapidly weaponized. Keeping systems patched and disabling legacy protocols (like SMBv1) is critical defense.
